KifuViewer

JavaScript で動く将棋の棋譜再生ライブラリです。

特徴

  • Vanilla JS
  • 高速 & 軽量 (21KB)
  • シンプル & カスタマイズ可能な UI
  • 少ない外部依存
    • Material Icons + Bootstrap のみ
    • カスタマイズで依存ゼロに変更可能
